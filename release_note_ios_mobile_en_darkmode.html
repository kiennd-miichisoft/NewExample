Lunascape v12.0.0 is ready

Save your precious time/battery with the new Lunascape, which supports dark mode and more efficient ad filters.

New Features

Dark Mode

Support for dark mode to save more battery power.

AD Blocker

The ad-blocking feature has been improved and can now be turned on or off on a per-website basis. This feature allows users to turn off the ad blocker only on pages where they do not wish to block ads or where the ad blocker would malfunction.

Address/SearchBar

URL Autocomplete and search suggestion functions have been enhanced.

Web3 Support

Experimental Web3 support feature is now available. When this feature is turned on from the settings screen, you can experience DApps on websites compatible with Ethereum-compatible networks.

Bug Fixes

Fixed video playback problems

Fixed a problem that video is automatically started on some sites.

Fixed video playback problems

Other minor problems have been fixed.
